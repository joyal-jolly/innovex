BLIND CODING
(only for school students)

January 23, 2026

A unique competitive coding challenge that tests participants' accuracy, memory, and typing skills. Participants must type a given code without viewing the monitor, focusing on logic recall and precision rather than debugging on-screen.

Event Rules & Guidelines

  • 1 A printed paper containing the code will be provided in each round.
  • 2 The monitor will be switched off once typing begins.
  • 3 The code must be written exactly as given.
  • 4 The program must be typed in the specified programming language only.
  • 5 No external help (mobile phones, internet, books, notes) is allowed.
  • 6 Errors will be counted after each round, The participant with the least number of errors will be declared the winner.
  • 7 The decision of the judges will be final and binding.
